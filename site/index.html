ToPS

Toolkit for probabilistic model of sequences

ToPS is an objected-oriented framework implemented using C++ that facilitates the integration of probabilistic models for sequences over a user defined alphabet.

Download

Features

Probabilistic models

ToPS contains the implementation of eight type of models to analyze discrete sequences.

Tools

This framework includes a set of programs that implement many commom operations.

Easy configuration

Users can define models either by manual description of the model, or by using training algorithms.

Performance

Written in C++, ToPS is really fast! It's been used to analyse huge amount of genomic data.

Extensible

ToPS is an extensible system that facilitates the implementation of other probabilistic models.

Wrappers

ToPS provides an C interface that can be accessed by other languages like python, ruby and R.
